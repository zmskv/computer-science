# CommentTree

## Описание

Комментарии — важная часть форумов, блогов, соцсетей и маркетплейсов. Чаще всего они имеют древовидную структуру: пользователи отвечают друг другу, а цепочки обсуждений уходят вглубь. Такая модель требует поддержки вложенности, удаления поддеревьев, сортировки, пагинации и поиска.

В этом задании нужно реализовать сервис для работы с древовидными комментариями с неограниченной вложенностью и дополнительными возможностями навигации.

## Основные требования

- Поддержать древовидную структуру комментариев с неограниченной вложенностью.
- Реализовать постраничный вывод и сортировку комментариев.
- Добавить полнотекстовый поиск по комментариям.
- Обеспечить удаление комментария вместе со всеми вложенными ответами.

## HTTP API

- `POST /comments` — создать комментарий, в том числе с указанием родительского комментария.
- `GET /comments?parent={id}` — получить комментарий и все вложенные ответы.
- `DELETE /comments/{id}` — удалить комментарий и все вложенные под ним ответы.

## Интерфейс

- Просмотр дерева комментариев с визуальной вложенностью, например за счет отступов.
- Создание новых комментариев и ответов.
- Удаление комментариев.
- Поиск комментариев по ключевым словам.

## Дополнительно
